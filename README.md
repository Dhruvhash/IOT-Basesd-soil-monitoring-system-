# Smart Soil Monitoring System

An IoT-based smart irrigation system that monitors soil conditions in real-time and automates irrigation using cloud-based dashboards and data logging.

---

## 🚀 Features

- 🌡️ Temperature & Humidity Monitoring  
- 🌱 Soil Moisture Detection  
- 🌧️ Rain Detection (Auto irrigation stop)  
- 👀 Motion Detection (Security monitoring)  
- 💧 Automated Irrigation (Relay-controlled pump)  
- 📱 Arduino Cloud Dashboard (Real-time control + visualization)  
- 📊 Google Sheets Integration (Data logging & analytics)  

---

- Soil Moisture Sensor

| Component | Pin |
|----------|-----|
| Soil Sensor | A0 |

| Relay Module | D3 |
| Push Button | D7 |

## ⚙️ Arduino Cloud Setup Guide

### 1. Create Account
- Go to Arduino IoT Cloud  
- Sign in / Sign up  

### 2. Create Thing
- Click **Create Thing**  
- Name: `Soil Monitoring System`  

### 3. Add Device
- Select ESP8266  
- Connect NodeMCU via USB  
- Upload auto-generated sketch  

### 4. Create Variables

| Variable | Type | Purpose |
|----------|------|--------|
| temperature | float | Temperature data |
| humidity | float | Humidity data |
| soilMoisture | int | Soil moisture |
| rainDetected | bool | Rain status |
| motionDetected | bool | Motion detection |
| relayControl | bool | Pump control |
| pirEnabled | bool | Motion toggle |

### 5. Build Dashboard

Add the following widgets:

- Gauges → Temperature, Humidity, Soil Moisture  
- LEDs → Rain Status, Motion Detection  
- Switch → Relay Control (Pump)  
- Switch → PIR Enable  

### 6. Upload Firmware

All logic, sensor handling, cloud communication, and automation is implemented in the **firmware files included in this repository**.

## 📊 Google Sheets Integration Guide

### 1. Create Google Sheet
- Create a new spreadsheet  

### 2. Open Apps Script
- Extensions → Apps Script  

### 3. Deploy Web App
- Deploy as **Web App**  
- Set access to **Anyone**  
- Copy generated URL  

### 4. Firmware Integration

The Google Sheets logging functionality is already implemented inside the **firmware**.  
Replace the Web App URL in the firmware before uploading.

ESP8266 → Google Sheets (Data Logging)

## ⚙️ Working Principle

- Sensors collect environmental data  
- ESP8266 processes and sends data to cloud  
- Dashboard displays real-time values  
- User can control pump remotely  
- System can automate irrigation based on conditions

## 🧪 Troubleshooting

| Issue | Solution |
|------|--------|
| No data update | Check WiFi & cloud connection |
| Incorrect readings | Calibrate sensors |
| Pump not working | Check relay & power supply |
